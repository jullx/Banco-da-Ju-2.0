# 🏦**Banco da Júlia 2.0**

Bem-vindo(a) ao **Banco da Júlia**, o banco mais confiável, rápido e simples do Brasil! 🇧🇷  
Esse é um projeto didático em Python que simula um sistema bancário no terminal.
---

# ✨ **Funcionalidades**

🔍 Verificar saldo  
💸 Realizar depósitos  
🏧 Efetuar saques (com validação)  
🚪 Sair do sistema  

---
